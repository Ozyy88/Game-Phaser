=== WATER HOP GAME ASSET GUIDE ===

Untuk memasukkan aset buatan Anda sendiri ke dalam game ini, letakkan file aset di folder 'assets/' ini dengan nama dan format berikut:

1. KARAKTER (Player Images):
   - assets/character_idle.png  -> Pose berdiri saat bersiap melompat.
   - assets/character_jump.png  -> Pose melayang ke atas ketika melompat.
   - assets/character_fall.png  -> Pose melayang turun ketika melompat.
   - assets/character_drown.png -> Pose saat tenggelam di air.

2. PIJAKAN (Platform Image):
   - assets/platform.png        -> Batang kayu, daun teratai, atau batu.

3. ITEM KOIN & JAM WAKTU (Collectibles):
   - assets/coin.png            -> Gambar koin emas (Poin).
   - assets/time.png            -> Gambar jam cyan (Booster Waktu +8s).

4. LATAR BELAKANG (Parallax Scenery):
   - assets/bg_sky.png          -> Langit latar belakang.
   - assets/bg_hills.png        -> Bukit/pepohonan tepi sungai.

Game ini telah dilengkapi dengan sistem pendeteksi otomatis. Jika file di atas ditambahkan ke dalam folder ini, game secara otomatis akan mendeteksi dan menggunakannya. Jika file belum ditambahkan, game akan menggunakan visual vektor (programmatic vector drawing) bawaan yang sudah dianimasikan dengan sangat mulus dan cantik!
